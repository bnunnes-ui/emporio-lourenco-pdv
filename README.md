# Empório Lourenço — PDV

Sistema de ponto de venda que funciona diretamente no navegador, sem instalação e sem servidor.

## Como usar

1. Abra `index.html` no Chrome, Edge ou Firefox.
2. Selecione os produtos. Itens vendidos por quilo permitem informar o peso.
3. Pressione **F10** ou clique em **Finalizar venda** e escolha o pagamento.
4. Use as áreas **Estoque** e **Vendas** no menu lateral.

Produtos, estoque e vendas ficam salvos no armazenamento local do navegador.

## Recursos

- Venda por peso e por unidade
- Busca e filtro por categoria
- Carrinho, desconto e quatro formas de pagamento
- Baixa automática do estoque
- Cadastro e edição de produtos
- Alertas de estoque baixo
- Histórico, faturamento e ticket médio
- Exportação de vendas em CSV
- Layout responsivo

Para uso real em vários caixas, recomenda-se uma segunda etapa com banco de dados, autenticação, backup e emissão fiscal.
